Instructions:

  • Use 'up', 'down', 'left', 'right' to move
  • Use 'pickup' to pick up the key
  • Use 'unlock' to open the door
  • Use 'for [number] [command]' to repeat a command, e.g., 'for 3 right'
  • Enter one command per line and press Enter to execute
  • Commands are executed in real-time after pressing Enter
